# kroman js

Kroman is a Korean hangul romanization tool.

This is the javaScript version.

It works in both server end and browser.

## Usage

``` javaScript
kroman.parse("손목시계")
=> 'son-mog-si-gye'
```

## Installation

Install it with npm:

```
npm install kroman
```
